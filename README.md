## 🎮 How to Play

1.  Run the main application file located in the `src` folder.
2.  Players take turns marking a square in a 3x3 grid with their respective symbols (X or O).
3.  The first player to get three of their symbols in a row (up, down, across, or diagonally) is the winner.
4.  If all squares are filled without a winner, the game ends in a draw.
